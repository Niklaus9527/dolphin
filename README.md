#海豚项目 -- Dolphin🐬

#项目目的

现在越来越多的项目都是基于hadoop、hive和spark等大数据类的项目，数据源的抽取和复杂业务逻辑计算都放到了大数据平台开发，在此过程中由于对大数据框架的不熟悉导致业务开发时间过长，海豚项目（Dolphin🐬）就是基于此原因，完善一套从数据源到大数据平台，并且基于配置化的、业务分离的一个组件，实现对存放在hdfs上面的文件进行大数据计算。

#计算流程
从关系型数据库到hdfs再到外部表

#数据源
* TXT（CVS） 文件
* parquet 文件
* MySQL 数据库
* Oracle 数据库

#导出文件
* TXT（CVS） 文件
* parquet 文件
* MySQL 数据库
* Oracle 数据库
